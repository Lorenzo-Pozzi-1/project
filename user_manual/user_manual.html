üìÖ Season Planner Page.

The Season Planner allows you to create spray plans from scratch, or starting from the plan of a previous year. It is a bit more complex than the rest of the program, but also more powerful. The season planner is divided into two main pages:

  • Scenarios Tabs Page: where you can create and manage your scenarios.
  • Scenarios Comparison Page: where you can compare the scenarios you imported/modified/created.

Scenarios Tabs Page

1. üóÇÔ∏è Scenarios tabs

Just like in Excel, you can drag to reorder, and double click to rename them!

2. üéÆ Scenarios Actions Buttons

Use them to manage the scenarios:

  • Import Scenario: Import a scenario from an Excel file. More on this in its own section.
  • New Scenario: Create a new, blank scenario tab.
  • Clone Current: Create a copy of the current scenario.
  • Delete: Delete the current scenario.
  • Compare Scenarios: Switch views to the comparison page for all the existing scenario tabs.
  • Export: Export an Excel version of your scenarios: each tab will become an individual worksheet in the Excel.

3. ‚úçÔ∏è Metadata Row

Use this row to fill in your grower-field-variety info. When importing a scenario, this row gets pre-compiled for you, and you can always change it. The area you set will be automatically applied to each new application.

4. üìã Applications Table

This is the container for all the applications: each row represents one and carries its information. The rows are smart: they will automatically detect if something's wrong and let you knwo by lighting up of a different colour, and calculate the EIQ score for you. If you're not sure why a row is lit up, just hover on it with the mouse and a little text will pop up telling you what's going on!

5. üéÆ Applications Actions Buttons

Use them to insert or remove an application row. Rows will always be added right below the one you have selected.

6. üìä Results

This section will tell you the name, number of applications, field EIQ score, where you are in the RegenAg Framework, and if something's wrong for the current scenario.

Scenarios Comparison Page

Here you can compare the scenarios you created/imported/modified in the Scenarios Tabs Page. The Scenarios will be displayed in a table format, with the applications grouped by type, and ordered by contribution to the overall Field Use EIQ score. Bounce between this and the tabs page to see what products you can switch or what rates you can change to improve your EIQ score.

Importing Scenarios

To import a scenario, start by clicking the "Import scenario" button. A pop-up will appear, prompting you to pick a file from your computer, remember: the file shoudl either be one auto-generated by [GXCore > Pesticide report > Farm Assurance Application Report > filter by crop-year, country, plant, grower, McCain field name] or one coming from this same app. For now, this app can only read these two formats.

Once selected, the file will be parsed, and you'll see a little summary, plus a pop-up if any product was not recognised in the database. If you see a lot of problems, chances are you selected a region in your preferences, and imported an application report for a different region, so many products could not be matched.

You'll be automatically brought to a second tab where you can fix the matching issues by either skipping them (e.g. adjuvants that do not contribute to EIQ), importing them as-is (you'll see the text, but the row will be excluded from the EIQ calculation), or map it to an existing product searching it by name. The row at the bottom of the pop-up will tell you what's going to happen when you give the "OK".

Once you solved all the product-matching issues, you'll see your applications. If any mistake filtered through when the applications were registered, now you'll see the row light up and be marked as "Incomplete" or "Invalid". If you're not sure why, just hover on the row with the mouse and a little text will pop up telling you what's wrong.

Once all the applications are rectified, you can rename the imported scenario by double clicking the tab name, and then clone it to create a new scenario based on it. Using the "Compare scenarios" button,you can bounce back and forths between views to see what you can do to reduce the EIQ score, or improve/adapt the plan to your grower's needs.
